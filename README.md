# Deanlist Portal

A Next.js + Tailwind student portal concept inspired by early-2000s university intranet design.

## Features
- Enrollment letter generator with dynamic name and student ID inputs
- Registrar merch shop
- Syllabus presented as a tracklist
- Hidden Dean's Office link

## Stack
- Next.js
- Tailwind CSS
- TypeScript

## Run locally
```bash
npm install
npm run dev
```
